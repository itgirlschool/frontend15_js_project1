# Описание структуры
--

## Список папок

* html - папка для верстки, каждая страница -- отдельная страница приложения, соединять их между собой пока не нужно
* css - папка для стилей, туда кладем стили с таким же названием, что и относящиеся к ним html
* fonts - шрифты
* images - картинки
